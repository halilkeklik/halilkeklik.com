Hugo ile oluşturduğum web sitesi

Hugo ile nasıl static web sitesi nasıl oluştuduğumdan anlatıcağım. İlk olarak google domainsten bir domain adresi aldım. Bu domain adresinin DNS ayarlarını kontrol etmek için cloudflare yönlendirdim. Domain işlerimi hallettikten sonra site oluşturma kısmı halletmem gerekiryordu. Bunun için Hugodan bir tema seçmem gerekti. Tema işini hallettikten sonra Hugonun kendi sitesinde ki tutorial kısmını takip ederek bir static site oluturmuş oldum. Oluturmuş olduğum bu site local bilgisayarımda çalışıyordu.Burada github pages devreye giricek.
